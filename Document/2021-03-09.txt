Redux + RxJs = NgRx

NgRx (Angular Reactive Extensions) :
	NgRx is Set of Libraries that uses the Redux Pattern and Reactive extentionn to solve State Management Problem.

State Management Problem Like :
			Request the same data from the API again
			Why is the modified data not reflected in the parent component?

Services solved problem of State Management System

NgRx is is a set of Angular libraries which includes:
					@ngrx/store
					@ngrx/effects
					@ngrx/router-store
					@ngrx/entity
					@ngrx/schematics
